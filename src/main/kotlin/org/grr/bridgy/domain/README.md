# domain 패키지

비즈니스 도메인 모듈입니다. 각 도메인은 `entity`, `repository`, `service`, `controller`, `dto` 계층으로 구성됩니다.

## 구조

```
domain/
├── chat/            # 카카오톡 대화 내역
├── customer/        # 고객 (카카오 유저)
├── reservation/     # 예약 관리
├── review/          # 리뷰 + AI 자동 답글
└── store/           # 매장 관리
```
